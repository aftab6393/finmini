## 🛠️ Tech Stack

**Backend:**
- Node.js + Express.js
- MongoDB (Mongoose ODM)
- JWT Authentication
- Socket.io (Real-time)
- Multer (File uploads)

**Frontend:**
- React + Vite
- Tailwind CSS
- Chart.js + react-chartjs-2
- React Router DOM
- Axios (API calls)

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- MongoDB running on localhost:27017

Backend
cd backend
npm install
npm run seed # Seeds products and demo users
npm run dev # Port 5000

Frontend
cd frontend
npm install
npm run dev # Port 5173
